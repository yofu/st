PLANE

format:

PLANE node1 node2 node3 min max

argument:

name type default  
node1 <int> * 一点目の節点番号
node2 <int> * 二点目の節点番号
node3 <int> * 三点目の節点番号
min <float> * Z座標の最小値
max <float> * Z座標の最大値

memo:

三点を通る平面から指定した距離に含まれる部分のみを表示する。
node1からnode2へのベクトルとnode2からnode3へのベクトルの
外積の方向を正方向とする。

link:

X, Y, Z, X-, Y-, Z-, CIRCLE
