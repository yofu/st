TIPS

  1. COMMONの中で定数や変数を定義して、ページ番号の指定や表示範囲の変更を楽にする。
    COMMON
    IOTA k
    CONST chapter 2.2
    CONST 2f 3.9 4.1
    PAGE a4tate $chapter.$k #ページ番号が2.2.1になる
    ELEM GIRDER
    Z $2f #Z=3.9からZ=4.1の範囲だけ表示する

  2. periodは長期の場合L、X方向地震時の場合X、Y方向地震時の場合Y、
    座屈解析の場合Bで指定し、増分解析の場合は「拡張子@ステップ数」で指定する。

  3. st.exeで偏心率・剛性率を求めるときに使用している水平剛性と部材変形角はそれぞれSTIFFとDRIFTで
    表示できる。
    例えば、1Fの偏心率の偏りの原因を調べるとき、
    PAGE a4tate
    ELEM column
    FENCE z 1
    STIFF X
    とすると、Z=1を通過する柱の水平剛性を表示できるので、偏心率を大きくしている原因の部材が見て取れる。

  4. 1F柱と2F梁のみ表示したい場合には、
    PAGE a4tate
    ELEM column girder
    Z 3 4
    FENCE+ z 1
    のように、FENCE+を使う。

  5. kjnファイルに軸組図表示用の基準線を定義しておき、
    名前を_X1のようにアンダーバー＋基準線名としておくと、
    AXONOMETRIC
    KIJUN ^_X
    Y1
    のようにして、Y1通り軸組図上にX方向通り芯を表示できる。
    kjnファイルの例>
    X1 0.000 -2.000 0.000 0.000 -1.000 0.000
    _X1 0.000 0.000 -2.000 0.000 0.000 -1.000
    このとき、通常の基準線のみを表示するには
    KIJUN !^_
    のように指定する。
