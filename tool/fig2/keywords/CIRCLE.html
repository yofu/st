CIRCLE

format:

CIRCLE x y radius distance

argument:

name type default  
x <float> * 円の中心のX座標
y <float> * 円の中心のY座標
radius <float> * 円の半径
distance <float> * 距離または[IN, OUT]

memo:

指定した円を底面とする円柱面を基準にして、
してした範囲に含まれる部分のみ表示する。
CIRCLE 10.0 20.0 5.0 IN
と指定すると、円柱の内側のみを表示し、
CIRCLE 10.0 20.0 5.0 OUT
と指定すると、円柱の外側のみを表示する。

link:

X, Y, Z, X-, Y-, Z-, PLANE
