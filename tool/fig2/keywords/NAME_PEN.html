NAME_PEN

format:

PEN名 elem enum …
PEN名 etype type …
PEN名 node nnum …
PEN名 sect snum …
PEN名 fence direction value
PEN名 error threshold
PEN名 yieldratio ratio

argument:

name type default  
elem     部材番号で対象を指定
enum <int> … 部材番号
etype     部材タイプで対象を指定
type <選択> … 部材タイプ [COLUMN, GIRDER, BRACE, WALL, SLAB, WBRACE, SBRACE]
node     節点番号で対象を指定
nnum <int> … 節点番号
sect     断面番号で対象を指定
snum <int> … 断面番号
fence     FENCEと同じ方法で対象を指定
direction <選択> * 方向 [X, Y, Z]
coord <float> * 座標
error     検定比で対象を指定
threshold <float> 1.0 検定比の閾値
yieldratio     塑性率で対象を指定
ratio <float> 4.0 塑性率の閾値

memo:

指定した対象を、PEN名の線種で描画する。
線種を予め指定しておく必要がある。
PEN RED 0.5 255 0 0
RED elem 1001 1002
RED etype column girder
RED node 101 102
RED sect 201 501
RED fence z 1
RED error 0.9
RED yieldratio 3.5
