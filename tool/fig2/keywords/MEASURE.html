MEASURE

format:

MEASURE n1 n2 direction offset overwrite dotsize rotate
MEASURE kijun k1 k2 offset dotsize rotate overwrite

argument:

name type default  
n1 <int> * 始端の節点番号
n2 <int> * 終端の節点番号
direction <選択> * 寸法線の引き出し方向 [X, Y, Z]
offset <float> * 寸法線の引き出し寸法
overwrite <string>   寸法の上書き値
dotsize <float> 0.1 黒丸の大きさ
rotate <float> 0.0 寸法値の回転角
kijun     基準線を使って寸法を指定する

memo:

寸法値を記入する。
上書き値が指定されない場合は、始端と終端の距離または基準線の始端同士の
距離を表示する。
MEASURE 101 102 X 0.1
MEASURE kijun X1 X2 0.1
のように、オフセット値まで指定すれば描画可能で、
途中のオプション指定を飛ばしたい場合には、
MEASURE 101 102 X 0.1 _ 0.5 90
のように、アンダーバーを書いておく。
ちなみに、overwriteの値はアンダーバーが距離で置換されるので、
MEASURE 101 102 X 0.1 @_
としておくと、寸法値が@500のように表示される。
overwriteの位置にNoneと書いておくと、寸法値が表示されない。
