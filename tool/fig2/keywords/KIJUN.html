KIJUN

format:

KIJUN string …

argument:

name type default  
string   … 表示する基準線の絞込み, Z方向への移動

memo:

kjnファイルで定義した基準線を表示する。
下記の記法で表示する基準線の絞込みが可能。

記法   絞込み方
^* : 指定した文字で基準線名が始まるものを表示する
!^* : 指定した文字で基準線名が始まるもの以外を表示する

例えば、
KIJUN ^X
とすると、X1やX2のようにXで始まる基準線のみを表示に追加し、
KIJUN !^Y
とすると、Y1やY2のようにYで始まる基準線以外を表示に追加する。

また、
KIJUN Z=5.0
のように書いておくと、そのページのみ基準線の始端と終端の
Z座標を指定した数値に変更する。
KIJUNMOVEを用いた場合は変更が次ページ以降にも継承される。
