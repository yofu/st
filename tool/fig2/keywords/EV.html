EV

format:

EV name format function

argument:

name type default  
name <string> * 表示データ名
format <string> * データ出力用フォーマット
function <string> * データ計算用の関数

memo:

functionで定義した任意の関数の返り値をformatで指定した
フォーマットで出力する。
例>
断面番号が101,102,105,111のものは柱幅0.3m、その他のものは柱幅0.2mとして
柱の小径を1/12.3のように分数表記で分母を小数点第1位まで表示する。
ELEM column
EV 柱の小径 "1/{:.1f}" x.length()/(0.3 if x.sect in [101,102,105,111] else 0.2)
