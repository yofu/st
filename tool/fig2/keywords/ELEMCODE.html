ELEMCODE

format:

ELEMCODE max
ELEMCODE error threshold
ELEMCODE num enum …

argument:

name type default  
max     各断面の検定比最大部材のみ表示
error     検定比がthreshold以上の部材のみ表示
threshold <float> 1.0 検定比の閾値
num     指定した部材番号のみ表示
enum <int> … 部材番号

memo:

部材番号を表示する。
ELEMCODE max
で検定比最大部材のみ、
ELEMCODE error 0.71428
で検定比が0.71428以上の部材のみ、
ELEMCODE num 1001 1002
で部材番号1001と1002のみをそれぞれ表示する。
