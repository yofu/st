STRESS

format:

STRESS etype period stype

argument:

name type default  
etype <選択> … 部材タイプ [COLUMN, GIRDER, BRACE, WALL, SLAB, WBRACE, SBRACE]
period <string> * 荷重時
stype <選択> … 応力タイプ [N, QX, QY, MT, MX, MY, SIGMA, TAU]

memo:

部材応力を表示する。
例えば長期柱軸力であれば、
STRESS column l n

応力タイプSIGMA, TAUは面材のときのみ有効で、
面材の直応力度とせん断応力度を表示する。
