FIGURE

format:

FIGURE pagesize pagenumber escape

argument:

name type default  
pagesize <選択> * ページサイズの指定 [a4tate, a4yoko, a3tate, a3yoko]
pagenumber <string>   ページ番号
escape     commonの内容を反映しない

memo:

応力図出力用のPAGE。
ページ内にSTRESSの行を含んでいる必要があり、
STRESS column l n qy mx
のように、軸力→せん断力→曲げモーメントの順に表示したい応力名を
記述する。
ページサイズがa4yoko、a3yokoの場合は左から右の順に、
ページサイズがa4tate、a3tateの場合は上から下の順に、
軸力、せん断力、曲げモーメントの図を一枚のpdfに書き出す。

FIGURE a4tate 7.2.1
STRESS column l n qy mx
...
と、
MULTIPLE 3 1
PAGE a4tate 7.2.1
STRESS column l n
...
PAGE a4tate
STRESS column l qy
...
PAGE a4tate
STRESS column l mx
...
はほぼ同じ。
