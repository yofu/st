FENCE

format:

FENCE direction coord
FENCE kijun coord
FENCE n1 n2 n3 coord
FENCE+ direction coord

argument:

name type default  
direction <選択> * 方向 [X, Y, Z]
kijun <string> * 基準線名で方向を指定
n1,n2,n3 <int> * 節点番号で方向を指定
coord <float> * 座標

memo:

指定した方向と座標の平面を横切る部材のみ表示する。
FENCE X1 0.1
のように基準線名で指定した場合、(X1通りの方向)×(Z軸方向)の外積方向を
座標軸としてcoord位置を横切る部材を選択する。
節点番号で指定する場合はPLANEと同様。
FENCEを用いると絞込み、FENCE+を用いると追加を行う。
