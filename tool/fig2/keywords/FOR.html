FOR

format:

for x in [x1,x2,x3,…]
…
endfor

  1. for x in [x1,x2,x3,…]
  2. for num,x in #N[x1,x2,x3,…]
  3. for v,w in [v1,v2,v3,…]x[w1,w2,w3,…]
  4. for num,v,w in #N[v1,v2,v3,…]x[w1,w2,w3,…]
  5. for n in #M~N

memo:

FORとENDFORの間に記述した内容を複製する。
その際、内部にある「$+変数名」はx1,x2,x3…
で置換される。
例えば、
for etype in [column,girder,wall,slab]
PAGE a4tate 2.2.1
ELEM $etype
endfor
は
PAGE a4tate 2.2.1
ELEM column
PAGE a4tate 2.2.1
ELEM girder
PAGE a4tate 2.2.1
ELEM wall
PAGE a4tate 2.2.1
ELEM slab
に展開される。

forの指定の仕方には上記5通りがある。

  • #Nを用いるとNから始まる数字が変数の先頭に追加される。
    Nを省略した場合は#1と同じ。
    for num,etype in #2[column,girder,wall,slab]
    PAGE a4tate 2.2.$num
    ELEM $etype
    endfor
    ↓
    PAGE a4tate 2.2.2
    ELEM column
    PAGE a4tate 2.2.3
    ELEM girder
    PAGE a4tate 2.2.4
    ELEM wall
    PAGE a4tate 2.2.5
    ELEM slab
  • *(アスタリスク)で複数の配列をつなげると、
    それぞれの配列から要素を１つずつ取り出して変数に展開する。
    つなげる配列は要素数が同じである必要があり、
    いくつでもつなげられる。
    先述の#Nと同時に使用することも可能。
    for num,etype,name,phi in #2[column,girder,wall,slab]*[柱,梁,壁,床]*[20,60,20,60]
    PAGE a4tate 2.2.$num
    ANGLE $phi -120
    TITLE $nameのみ表示
    ELEM $etype
    endfor
    ↓
    PAGE a4tate 2.2.2
    ANGLE 20 -120
    TITLE 柱のみ表示
    ELEM column
    PAGE a4tate 2.2.3
    ANGLE 60 -120
    TITLE 梁のみ表示
    ELEM girder
    PAGE a4tate 2.2.4
    ANGLE 20 -120
    TITLE 壁のみ表示
    ELEM wall
    PAGE a4tate 2.2.5
    ANGLE 60 -120
    TITLE 床のみ表示
    ELEM slab
  • #M~Nで指定すると、MからNまでの整数を順次変数に代入する。
