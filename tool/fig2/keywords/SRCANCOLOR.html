SRCANCOLOR

format:

SRCANCOLOR
SRCANCOLOR lw width
SRCANCOLOR obscure

argument:

name type default  
lw     線幅を指定する
width <float>   線幅
obscure     検定比が設定されていない部材を細線表示

memo:

検定比に応じて色表示をする。
lwに続けて線幅を指定すると色表示した部材の線幅を変更できる。
この機能は
PEN SRCAN 0.1
のように、PENを使っても代用可能。
検定比が設定されていない部材は黒線で描画されるが、
obscureとした場合には線色が(R,G,B)=(128,128,128)のグレー、
線幅が0.05mmとなる。
