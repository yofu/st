ZMEASURE

format:

ZMEASURE n1 n2 direction offset overwrite dotsize rotate

argument:

name type default  
n1 <int> * 始端の節点番号
n2 <int> * 終端の節点番号
direction <選択> * 寸法線の引き出し方向 [X, Y, Z]
offset <float> * 寸法線の引き出し寸法
overwrite <string>   寸法の上書き値
dotsize <float> 0.1 黒丸の大きさ
rotate <float> 0.0 寸法値の回転角

memo:

Z方向の寸法値を記入する。
詳細はMEASUREの項を参照。
