NAME_KIJUN

format:

基準線名 min max noname reverse

argument:

name type default  
min <float> -0.1 表示範囲の最小値
max <float> 0.1 表示範囲の最大値
noname     基準線名を表示しない
reverse     視点の向きを反転する

memo:

指定した基準線とZ軸を含む平面上の部材を表示する。
表示範囲と視点を自動で設定するので、
軸組図のように表示するときに便利。
表示範囲はminとmaxで指定し、
X1 0.4 0.6
のように指定することで、X1+500通りを表示することも可能。
視点は表示平面に直交する方向から見て、基準線の始端から終端の方向が左から右になるように
設定され、reverseを指定すると逆転する。
また、デフォルトでPAGETITLEに基準線名が追加される。
これはnonameを指定することで回避できる。
